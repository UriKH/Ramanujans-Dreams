## Notation Conventions
| Symbol | Meaning | LaTeX |
|--------|---------|-------|
| $t_n$ | Summand of a Ramanujan series | `t_n` |
| $P_n$ | Partial sum $\sum_{k=0}^n t_k$ | `P_n` |
| $S_n$ | Tail sum $\sum_{k=n}^\infty t_k$ | `S_n` |
| $T_n$ | Stripped tail sum: $S_n = \Phi_n T_n$ | `T_n` |
| $\Phi_n$ | Pochhammer prefactor | `\Phi_n` |
| $C_n$ | Consecutive ratio $t_n/t_{n-1}$ | `C_n` |
| $r_k$ | Prefactor ratio $\Phi_k/\Phi_{k-1}$ | `r_k` |
| $\Theta$ | $z \frac{d}{dz}$ operator | `\Theta` |
| $U_N$ | $4\times4$ conjugation matrix | `U_N` |
| $R_T(n)$ | $2\times2$ $T_n$ recurrence matrix | `R_T(n)` |
| $M_P(n)$ | $2\times2$ $P_n$ companion matrix | `M_P(n)` |
| ${}_pF_q$ | Generalized hypergeometric function | `{}_pF_q` |
| CMF | Conservative Matrix Field | `CMF` |

## Citation Style
- Use `biblatex` with `biber` backend, `numeric-comp` style.
- Cite Ramanujan's 1914 paper as `\cite{ramanujan1914}`.
- Cite the Ramanujan Machine papers as `\cite{raayoni2021ramanujan}`.
- Cite CMF papers as `\cite{cmf2024}`.
- When adding new references, add to `references.bib` and use a descriptive key: `authorYEARkeyword`.

## Writing Guidelines
- Two-column format, 10pt, `article` class.
- Keep sections modular ‚Äî one file per section.
- Use `\cref{}` for cross-references (figures, tables, equations, sections).
- Define new math operators in `config.tex`, not in section files.
- Use `booktabs` for tables (`\toprule`, `\midrule`, `\bottomrule`).
- Placeholder figures: use `\placeholderfig{description}{label}{caption}`.
- TODO markers: use `\TODO{text}` ‚Äî highlighted in red.
